### BIG-IP APMOverview: Configuring step-up authenticationThis section describes the basic elements for how to configure a per-request policy for step-up authentication. You need to have created a per-session policy. You also need to have configured the authentication servers you plan to use and have added them to the BIG-IP system. The steps for adding the servers differ depending on the method you are using.
